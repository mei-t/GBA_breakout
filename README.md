# Breakout on GBA  
This is a breakout working on Game Boy Advance.

## Requirement  
Download devkitPro : <https://devkitpro.org/wiki/Getting_Started>


## How to compile  
Just do `$ make`.


## How to run  
Run `GBA_breakout.gba` with your favorite GBA emulator.
